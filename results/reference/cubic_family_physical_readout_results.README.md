# cubic_family_physical_readout_results.csv

Diese Datei ist kein wiedergefundenes Originalresultat, sondern eine minimale rekonstruktive Hilfstabelle.
Sie enthält nur die Modenselektionen (`primitive_mode`, `fcc_mode`) für β = 0, 1, 5,
die aus `results/side_branches/field_information/reference/field_information_distribution_supercell_test.csv`
zurückgelesen wurden, damit das historische Skript `field_information_distribution_supercell.py`
in der Repo-Umgebung wieder laufen kann.
