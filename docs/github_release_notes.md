# GitHub release notes

Recommended first public release: **v1.0.0**

This repository is intended as the reproducibility backbone for the companion paper:

**Artifact-Hardened Ordering in a 3D Helmholtz-Robin Eigenproblem on Cube-Like and Lattice-Linked Geometries**

## What the public release should contain

- the curated code under `src/`
- runnable wrappers under `scripts/`
- versioned reference outputs under `results/`
- explanatory material under `docs/`
- provenance material under `archive/`

## What not to add through the GitHub UI

Do not initialize the remote repository with a new README, license, or gitignore. Those files are already part of this bundle and should remain authoritative.
