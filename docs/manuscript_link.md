# Manuscript linkage

This repository accompanies the manuscript:

**Reproducible Low-Mode Ordering in a 3D Helmholtz--Robin Eigenproblem on Cube-Like and Lattice-Linked Geometries**

Author: **Steven Trümpert**  
Contact: **StevenTruempert@hotmail.de**

Purpose of this repository:

- provide the numerical code used in the manuscript,
- provide runnable scripts for the reported analyses,
- provide reference outputs against which results can be checked,
- document the minimal remaining gaps or reconstructed pieces relevant to reproduction.

This repository is not intended to function as a standalone research diary.
