# How to cite this repository

Until the public DOI is minted, cite the repository by release tag together with the corresponding Paper 1 preprint.

Before public release, update:

- final author metadata in `CITATION.cff`
- repository URL
- Zenodo DOI
- arXiv identifier of the paper
